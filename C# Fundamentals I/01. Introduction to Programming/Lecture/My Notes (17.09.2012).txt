Visual Studio How To: 

1. In order to be able to debug / run projects within the same Solution by highlighting them in Solution Explorer and pressing Ctrl + F5 (F5) go to: 
Solution - (Right click) Properties - Common Properties - Startup Project - Current Selection

2. Create a new project within the same Solution 
Right click Solution - Add - New Project
or
File - Add - New Project

Visual Studio Shortcut Keys 

Indent block of code to the right: Highlight the block of code and press Tab
Indent block of code to the left: Highlight the block of code and press Shift + Tab
