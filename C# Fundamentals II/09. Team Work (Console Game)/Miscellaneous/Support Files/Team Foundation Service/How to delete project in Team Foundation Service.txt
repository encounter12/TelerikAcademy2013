Command:
1. Open command prompt in: C:\Program Files (x86)\Microsoft Visual Studio 11.0\Common7\IDE
2. Run one of the commands below: 

tfsdeleteproject.exe /force /collection:https://academytelerik.VisualStudio.com/DefaultCollection TeamSnoopy
TfsDeleteProject /collection:https://academytelerik.visualstudio.com/DefaultCollection TeamSnoopy
tfsdeleteproject.exe /force /collection:https://academytelerik.visualstudio.com/DefaultCollection TeamSnoopy

tfsdeleteproject.exe /q /force /collection:https://academytelerik.visualstudio.com/DefaultCollection TeamSnoopy
